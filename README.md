# YouTube Transcriptions

Public transcript archive generated by the private local `youtube-local-transcriber` project.

Web app:

```text
https://robert-fiedor.github.io/youtube-transcriptions/
```

This repository stores transcript artifacts only:

- `.txt` plain text transcripts
- `.srt` subtitle files
- `.vtt` subtitle files
- `.json` timestamped segment data and source metadata

Raw audio and video files do not belong in this repository.

## Layout

Each video lives under:

```text
transcripts/<youtube-video-id-or-generated-stem>/
```

The web app reads `data/transcripts.json`, which is generated from the transcript JSON files:

```bash
python3 scripts/build_manifest.py
```

## Secret Scanning

This repo includes a tracked pre-commit hook at `.githooks/pre-commit`. Enable it locally with:

```bash
git config core.hooksPath .githooks
```
